# All About Pets - Thesis Proposal Project

by: Soriano Kenny, Obina Aldrin, Brizo Daniella, Daguro Elijah


## Overview

This repository contains the source code for **All About Pets**, a veterinary clinic web system concept built with **Vue.js** and **Vite**.

Originally developed as a **thesis proposal with a real client**, this project is now **discontinued** and maintained as a portfolio showcase of UI/UX design, page flow, and front-end implementation.

The website includes:

- Public-facing clinic pages and services information
- User and admin login interfaces
- Registration and consultation form flows
- Admin dashboard prototype

## Project Status

- Status: **Discontinued Thesis Proposal**
- Client Context: **Based on a real client requirement during thesis development**
- Current Purpose: **Portfolio presentation and technical showcase**

## Features

- Responsive multi-page veterinary clinic website experience
- Vue-powered routing for legacy page navigation
- Preserved original thesis UI and content flow
- Admin dashboard mock workflow
- Consultation and rating page prototypes

## Tech Stack

- Vue 3
- Vue Router
- Vite
- HTML/CSS/JavaScript (legacy thesis pages preserved under `public/legacy`)

## Installation and Setup

```bash
# Clone the repository
git clone https://github.com/DevKennySoriano/allaboutpets

# Navigate into the project directory
cd allaboutpets

# Install dependencies
npm install

# Start development server
npm run dev
```

## Build for Production

```bash
npm run build
```
